Privacy Policy

This policy explains how Iva, a Discord bot, handles your personal information. By using Iva, you agree to this policy.

Information Collection and Use

Iva collects and stores:

  • User messages (encrypted)
  • OpenAI API keys (encrypted)
  • User settings

Third-Party Services

Iva uses third-party services:

  • Heroku for hosting
  • OpenAI API for language model completions

Data Storage and Retention

Iva keeps user data indefinitely unless the user requests deletion.

Data Access, Correction, and Deletion

Use these commands with Iva:

  • /reset: Deletes message history for that channel or thread.
  • /setup: To delete or replace the OpenAI API key, input an empty value.

Age Limitations

Iva is for users 13 years or older.

Changes to This Privacy Policy

We may update this policy and will post changes on this page.

Contact Us

For questions, contact us through our Discord support channel or email at hi@iva.gg.
