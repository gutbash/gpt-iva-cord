Privacy Policy

Introduction

This policy explains how Iva, a Discord bot, handles your personal information. By using Iva, you agree to this policy.

Information Collection and Use

Iva collects and stores the following data which is protected by AES-256 encryption at the content level and the server level at rest:

  • Message Content
  • API Keys

Data Storage and Retention

Message content data is deleted every 30 days, however can be deleted at any time upon request.

Data Access, Correction, and Deletion

Use these commands with Iva to delete all associated data:

  • /reset: Deletes all message content for that channel or thread.
  • /erase: Deletes all message content associated with a user from all channels and threads.
  • /setup: Deletes the API key entry associated with a user when no argument is given.

Age Limitations

Iva is for users 13 years or older.

Changes to This Privacy Policy

We may update this policy and will post changes on this page.

Contact Us

For questions, contact us through our Discord support channel or email at hi@iva.gg.
